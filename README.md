# Stack
Stack game clone made with Three.js

## Live
You can play now [here](https://konradlinkowski.github.io/Stack/)

![gameplay screenshot](./screenshots/screen.png)

## Installation
```
# install dependencies
yarn install
# run the game
yarn start
# alternatively run the dev server
yarn dev
```
